Vertical Partitioning

Vertical Partitioning involves splitting a table by columns.

For example, storing frequently accessed columns (username, status) in one table and heavy, rarely used columns (bio, profile_image_blob) in another.

User Table (Big)
ID | Name | Bio | Photo
